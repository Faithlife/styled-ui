# Text

✔ Styled-Systems Compliant  
✔ WAI ARIA Compliant

`Text` is a primitive layout component that can be used to style text with a predefined typographical style and to customize typography-related styling. By default, it renders as a `span`. `Paragraph` is also available for block text, and has the same API as `Text`.

## Example

<Canvas>
	<Story
		name="Text Example"
		args={{ textStyle: 'c.16', whiteSpace: 'pre-wrap' }}
		argTypes={{
			...resetArgTypes,
			whiteSpace: { table: { display: true } },
		}}
	>
		{args => (
			<Text {...args}>
				{`Text content using the `}
				<Text color="purple4" fontWeight="semibold">
					{args.textStyle}
				</Text>
				{` text style.`}
			</Text>
		)}
	</Story>
</Canvas>

## Available text styles

### Headings

Use for headlines.

<Stack spacing={3}>
	<ThemeList
		items={theme => theme.textStyles.h}
		render={([key, value]) => (
			<Box key={`h.${key}`}>
				<Paragraph textStyle={`h.${key}`}>h.{key}</Paragraph>
			</Box>
		)}
	/>
</Stack>

The `Heading` component is also available for convenience. It's a simple wrapper around `Parargaph`, rendering a `heading` element by default. Pass a `level` prop corresponding to the named `h.{level}` names listed above.

<Canvas>
	<Story
		name="Heading Example"
		args={{ level: 18 }}
		argTypes={{
			...resetArgTypes,
			textStyle: { table: { display: false } },
			level: { control: { type: 'select', options: headingLevelOptions } },
		}}
	>
		{args => <Heading {...args}>My Heading</Heading>}
	</Story>
</Canvas>

### Content

Use for body copy and miscellaneous content.

<Stack spacing={3}>
	<ThemeList

### UI

UI components like buttons. Line heights always match the font size to simplify spacing.
