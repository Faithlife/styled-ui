# Accordion

✔ Styled-Systems Compliant  
✔ [WAI ARIA Compliant](https://www.w3.org/TR/wai-aria-practices-1.1/#accordion)

This component does not directly support limiting that no more than one section can be open. However, this component uses a fully-controlled pattern and this functionality could be implemented with a wrapping component in consuming code.

An `<Accordion>` can have any number of `<Accordion.Item>` children.  
Each `<Accordion.Item>` should have exactly one `<Accordion.Header>` and one `<Accordion.Panel>` as children.

### With subtitles
